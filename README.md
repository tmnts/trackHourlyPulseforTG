# trackHourlyPulseforTG
Simple JS-solution to track your channel's ERR rate hourly via TGStat API
This script gets data from TG Stat API and put it into your Google Sheet. Extensions -> Apps Script
Don't forget to set the timer at Apps Script to 1 hour or less.
Suitable for lesser one-man channels approx 10000 subs with 10 posts per day.
